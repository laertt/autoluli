# Project Summary
**Auto Luli** is an Albanian e-commerce platform specializing in car parts, offering an extensive selection from 1980 to 2026. It aims to provide users with an intuitive shopping experience, complete with customer support through various channels.

# Project Module Description
The website consists of the following functional modules:
- **Home Page**: Displays a hero section, search functionality, and highlights available car parts.
- **Search Page**: Enables users to filter and search for specific car parts based on various criteria.
- **About Page**: Shares the store's history and values.
- **Contact Page**: Allows users to inquire via a contact form or direct messaging options, including links to Instagram and Google Maps.
- **Footer**: Contains quick links and contact information.

# Usage
To get started with the project, follow these steps:
1. Install dependencies:
   ```
   pnpm install
   ```
2. Build the project:
   ```
   pnpm run build
   ```
3. Run the application:
   ```
   pnpm run dev
   ```

## Hosting on GitHub Pages
1. Create a new repository on GitHub (e.g., `auto-luli`).
2. Upload your code:
   ```bash
   git init
   git add .
   git commit -m "first commit"
   git branch -M main
   git remote add origin https://github.com/USERNAME/REPO-NAME.git
   git push -u origin main
   ```
3. Enable GitHub Pages:
   - Go to **Settings → Pages** in your repository.
   - Under "Source," select **GitHub Actions**.
4. Create the `.github/workflows/deploy.yml` file with the specified workflow for deployment.

**Important**: If your repository is not named `username.github.io`, add `base: '/your-repo-name/'` in `vite.config.ts`.
